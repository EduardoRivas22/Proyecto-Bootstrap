Desarrollo del Proyecto
Diseño de una web estática para la "I Carrera Popular Solidaria" utilizando Bootstrap 5 y control de versiones.

Secciones y Columnas
- Navbar: Componente de navegación responsive.
- Hero: Cabecera de impacto con Flexbox.
- Información: Grid de 2 columnas (col-md-6).
- Categorías: Grid de 3 columnas usando el componente Cards de Bootstrap (col-md-4).

Historial de Commits
1. Estructura inicial: Creación del esqueleto HTML y secciones de contenido.
2. Sección Hero: Implementación del área principal de presentación.
3. Estilos propios: Vinculación de style.css para ajustes de espaciado.

Dificultad encontrada
La mayor dificultad fue lograr que las tarjetas (Cards) tuvieran la misma altura. Se solucionó utilizando la clase de Bootstrap h-100.

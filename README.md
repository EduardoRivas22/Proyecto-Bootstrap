# Carrera Popular Ciudad de Zaragoza 2024

## Descripción del Proyecto
[cite_start]Diseño de una web informativa para una carrera popular de 5K y 10K en el Parque del Agua[cite: 5, 7].

## Estructura de Bootstrap
* [cite_start]**Navbar:** Barra de navegación fija con enlaces internos[cite: 23].
* [cite_start]**Hero Section:** Cabecera con imagen de fondo personalizada mediante CSS[cite: 24, 31].
* [cite_start]**Sección Información:** Grid de 2 columnas (`col-md-6`) para texto y lista de detalles[cite: 25, 36].
* [cite_start]**Sección Recorrido:** Grid de 3 columnas (`col-md-4`) usando componentes de **Cards**[cite: 25, 27, 37].
* [cite_start]**Footer:** Pie de página con información de contacto[cite: 30].

## Historial de Commits
1. [cite_start]**Commit 1:** Estructura HTML base y enlace a CDN de Bootstrap[cite: 21, 38].
2. [cite_start]**Commit 2:** Creación de las secciones principales usando el sistema de rejilla (Grid)[cite: 25, 38].
3. [cite_start]**Commit 3:** Vinculación de estilos CSS propios para el Hero y finalización del Footer[cite: 31, 38].

## Dificultad Encontrada
La mayor dificultad fue asegurar que las Cards de la sección de recorrido mantuvieran el mismo tamaño en dispositivos móviles. [cite_start]Se solucionó utilizando las clases de espaciado `my-5` y sombras `shadow-sm` de Bootstrap para mejorar la legibilidad[cite: 39].
